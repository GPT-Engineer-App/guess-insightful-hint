# guess-insightful-hint

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/guess-insightful-hint.git
cd guess-insightful-hint
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
